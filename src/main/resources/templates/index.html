📌 API Endpoints
POST /api/donations Create
GET /api/donations List all
GET /api/donations/{id} By ID
GET /api/donations/available Available only
GET /api/donations/nearby?location=Mumbai 🌟 Location match
PUT /api/donations/{id}/claim Claim
PUT /api/donations/{id}/status Update status
DELETE /api/donations/{id} Delete
GET /api/donations/stats 📊 Statistics
📝 Example Request
📋 POST /api/donations 
{
    // Required fields
    "foodType":      "Rice, Vegetables, Bread",
    "quantity":       50,
    "location":       "Mumbai, Maharashtra",
    "donorName":      "Hotel Taj",
    "donorPhone":     "+91-9876543210",
    "expiryDate":     "2026-07-15",
    
    // Optional
    "description":    "Fresh cooked meals from hotel buffet"
}
🧪 Quick Test with cURL
📤 Create
curl -X POST http://localhost:8080/api/donations \
  -H "Content-Type: application/json" \
  -d '{"foodType":"Rice","quantity":50,"location":"Mumbai","donorName":"Hotel Taj","expiryDate":"2026-07-15"}'
📋 List All
curl http://localhost:8080/api/donations
✅ Claim
curl -X PUT http://localhost:8080/api/donations/1/claim
📊 Stats
curl http://localhost:8080/api/donations/stats
